# Student Mobile Catalog

## Module Summary
- Module ID: `MOD-09`
- Module Name: Student Mobile App
- Primary Domain: Mobile

## Function Catalog
| Function ID | Function Name | Brief Description |
|---|---|---|
| FUN-09-01 | Onboarding and welcome flow | Guide first-time users and route by role selection. |
| FUN-09-02 | Login and token persistence | Authenticate students and restore sessions securely. |
| FUN-09-03 | 4-step registration flow | Identity verification, account setup, face registration, review. |
| FUN-09-04 | Attendance dashboard and history | Show current status, schedule, and historical attendance. |
| FUN-09-05 | Profile and face re-registration | Manage profile fields and renew face data. |
| FUN-09-06 | Student notifications | Show alerts and realtime updates in student notification UI. |

## Screen Domains
- Shared app entry: `SCR-001`, `SCR-002`, `SCR-003`
- Auth and registration: `SCR-004`, `SCR-006`, `SCR-007`, `SCR-008`, `SCR-009`, `SCR-010`
- Student portal: `SCR-011` to `SCR-018`

## API Domain
Module 9 consumes APIs from auth, user, face, schedules, attendance, and websocket modules.
