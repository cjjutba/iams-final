# Output Contract

Every AI implementation task for MOD-09 must return:

1. Scope
- Implemented `FUN-09-*` IDs
- Explicitly excluded items

2. Changes
- Files modified/created with brief reason

3. Validation
- Tests executed (`T09-*`) and result summary
- Manual verification steps completed

4. Contract Compliance
- Confirmation consumed API contracts were followed
- Confirmation required screen IDs and UI states were implemented

5. Risks and Follow-Ups
- Known risks
- Required docs or decision follow-ups
