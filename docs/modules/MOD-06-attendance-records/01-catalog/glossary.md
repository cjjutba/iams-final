# Glossary

- Attendance Record: Daily status row for student in a schedule.
- Check-in Time: First detection time for attendance date.
- Presence Score: Percentage derived from scan detections (from presence module context).
- Manual Override: Faculty-provided status change with remarks.
- Live Attendance: Session-time roster state for class.
- Dedup Rule: one row per `(student_id, schedule_id, date)`.
