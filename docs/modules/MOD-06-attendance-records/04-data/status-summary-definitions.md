# Status and Summary Definitions

## Attendance Status Values
- `present`: detected on time and participated sufficiently
- `late`: first detection after grace period
- `absent`: never detected
- `early_leave`: left before class ended

## Summary Metrics
Typical summary block includes:
- total
- present
- late
- absent
- (optional) early_leave

## Consistency Rule
Summary counts must equal filtered record totals for the same query scope.
