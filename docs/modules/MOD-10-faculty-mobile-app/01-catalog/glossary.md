# Glossary

- Faculty stack: authenticated faculty-only navigation scope.
- Active class: schedule context currently in-session by day/time.
- Live roster: dynamic per-student attendance state for current class.
- Manual override: faculty-issued attendance correction via dedicated API.
- Alert feed: ordered list of early-leave and related warning events.
- Session restore: token/session hydration during app startup.
- Summary card: class-level status totals shown at session end.
- UI state triad: loading, empty, and error handling for each data view.
