# Realtime Catalog

## Module Summary
- Module ID: `MOD-08`
- Module Name: Realtime Notifications and WebSocket
- Primary Domain: Backend + Mobile

## Function Catalog
| Function ID | Function Name | Brief Description |
|---|---|---|
| FUN-08-01 | Open authenticated WebSocket connections | Accept socket only for authenticated user context. |
| FUN-08-02 | Publish attendance update events | Push attendance status changes to relevant clients. |
| FUN-08-03 | Publish early-leave events | Push early-leave alerts generated by presence logic. |
| FUN-08-04 | Publish session-end summary events | Push class-end summary payload to clients. |
| FUN-08-05 | Handle reconnect and stale cleanup | Recover from disconnect and clean stale connection state. |

## Event Catalog
| Event Type | Producer | Consumer |
|---|---|---|
| `attendance_update` | Attendance flow (`MOD-06`) via notification service | Faculty live view, notification screens |
| `early_leave` | Presence flow (`MOD-07`) via notification service | Faculty alerts/live view, notifications |
| `session_end` | Session finalization flow | Faculty and student notifications |

## API Surface
- `WS /ws/{user_id}`
