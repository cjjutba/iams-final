# Output Contract

Every AI implementation task for MOD-08 must return:

1. Scope
- Implemented function IDs (`FUN-08-*`)
- Explicitly excluded items

2. Changes
- Files created/modified with short rationale

3. Validation
- Tests executed (`T08-*`) and outcomes
- Manual verification steps performed

4. Contract Compliance
- Confirmation that event payloads match `03-api/event-*.md`
- Confirmation that auth and user-binding rules were enforced

5. Risks and Follow-Ups
- Outstanding risks
- Needed decisions or doc updates
